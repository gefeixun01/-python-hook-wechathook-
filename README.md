# -python-hook-wechathook-
使用纯python环境对微信进行hook，并调用微信功能，包括取sessionid，小程序code，云函数等

* 文档正在开发中，功能不断完善
* 该DLL非http接口，使用的是加载DLL,并调用函数的形式
* DLL可使用易语言，Python，c++等编程语言加载
* 本DLL可以用来开发框架
